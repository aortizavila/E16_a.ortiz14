Cortana es un asistente virtual creado por Microsoft para sus plataformas tanto de computadores personales, como plataformas de video juegos. Con cortana se pueden establecer recordatorios, reconocer voz natural, así como responder preguntas utilizando el navegador Bing. Cortana se basa en redes neuronales aplicadas para el procesamiento de lenguaje natural. Las redes neuronales, hoy en dia se alimentan de 4 plataformas que son usadas para el funcionamiento de cortana: Azure Stream Analytics, Azure HDInsight, Azure Data Lake Analytics, Azure Machine Learning. Azure Stream Analytics es un motor de procesamiento de eventos en tiempo real basado en la nube totalmente administrado que permite a las empresas obtener información sobre la transmisión de datos en tiempo real. Azure data lake analytics es un servicio de procesamiento de datos a gran escala en la Nube de Azure, diseñado específicamente y optimizado para cargas de trabajo analíticas, lo que simplifica el análisis de Big Data.  Cortana usa el modelo BERT (Bidirectional Encoder Representations from Transformers). BERT está diseñado para entrenar previamente representaciones bidireccionales profundas, condicionando conjuntamente el contexto izquierdo y derecho en todas las capas. Como resultado, las representaciones BERT pre-entrenadas pueden ajustarse con solo una capa de salida adicional para crear modelos de vanguardia para una amplia gama de tareas, como la respuesta a preguntas y la inferencia de lenguaje, sin tareas sustanciales. Modificaciones específicas de la arquitectura. BERT utiliza Transformer, un mecanismo que aprende las relaciones entre palabras (o sub-palabras) en un texto. Transformer incluye dos mecanismos separados: un codificador que lee la entrada de texto y un decodificador que produce una predicción para la tarea. Dado que el objetivo de BERT es generar un modelo de lenguaje, solo se necesita el mecanismo del codificador.
